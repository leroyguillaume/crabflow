# Contributing

## Requirement

- [Rust](https://rustup.rs/)
- [pre-commit](https://pre-commit.com/)

## Getting started

```bash
git clone https://github.com/leroyguillaume/crabflow.git
cd crabflow
pre-commit crabflow
```

## How to build

```bash
cargo build
```

## How to test

```bash
cargo test
```
