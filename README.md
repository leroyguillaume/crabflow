# crabflow

crabflow is a task orchestrator written in Rust.

## Getting started

### Rust

Add `crabflow` in your `Cargo.toml`:
```toml
[dependencies]
crabflow = "0"
```

### Python

Add `crabflow` in your `requirements.txt`.
